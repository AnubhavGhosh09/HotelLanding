# Stays Inn - Budget Hotel Website

A clean, responsive single-page hotel website built with semantic HTML5 and modern CSS3. Designed for budget-conscious travelers in India with accessible pricing and essential amenities.

## Features

- **Responsive Design** - Mobile-first approach with breakpoints at 768px, 1024px
- **Hero Section** - Full-viewport hero with animated entrance
- **Room Cards** - 6 room types with pricing from ₹899 - ₹2,899
- **Amenities Grid** - 5 key amenities with icons
- **Why Choose Us** - 4 trust-building features
- **Guest Reviews** - 3 testimonial cards
- **Location Section** - Contact info with nearby landmarks
- **Footer** - Quick links, contact details, social icons

## Tech Stack

- **HTML5** - Semantic markup (`<header>`, `<section>`, `<article>`, `<footer>`)
- **CSS3** - Modern features only (Flexbox, Grid, CSS Variables, Transitions)
- **No JavaScript** - All interactions via `:hover`, `:focus`, CSS animations
- **Google Fonts** - DM Sans + Playfair Display

## Quick Start

Simply open `index.html` in any modern browser.

```bash
# Using macOS
open index.html

# Using Linux
xdg-open index.html

# Using Windows
start index.html
```

## Sections

1. **Navigation** - Sticky header with logo and links
2. **Hero** - Full-screen intro with CTA
3. **Rooms** - 6-card grid with features and pricing
4. **Amenities** - 5-icon grid on dark background
5. **Why Choose Us** - 4 benefit cards
6. **Reviews** - 3 guest testimonial cards
7. **Location** - Two-column layout with map image
8. **Footer** - 3-column layout with social links

## Responsive Breakpoints

- **Mobile** (<768px): Single column layout
- **Tablet** (768-1024px): 2-column grids
- **Desktop** (>1024px): 3-column grids
